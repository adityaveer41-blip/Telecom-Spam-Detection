# Telecom Spam Detection Platform
> End-to-End ML + GenAI System for Voice Spam Detection

## Overview
An internship project to build a production-grade telecom voice spam detection system using ML and Generative AI. The system analyzes Call Detail Records (CDR) to identify spam callers with high precision and provides LLM-powered explanations for flagged numbers.
